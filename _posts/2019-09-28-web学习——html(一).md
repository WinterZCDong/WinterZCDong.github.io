## html 格式化标签

### pre 标签

​	预格式文本，pre标签中的文本通常会保留空格和换行符。文本会呈现为***等宽字体***。pre标签的一个常见应用就是显示计算机的 ***源代码***

​	部分标签不能放入pre标签定义的块中，如<p>和<address>

​	pre标签中的特殊符号将杯转换为符号实体，如下表

| 特殊符号 | 符号实体 |
| :------: | :------: |
|  \&gt;   |    >     |
|  \&lt;   |    <     |
